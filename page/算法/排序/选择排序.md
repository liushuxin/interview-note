# 选择排序

### 1、原理

每次找出数组中最大/最小元素，放到头部或者尾部就好了

### 2、具体实现

```javascript
function selectSort(array) {
  const len = array.length;
  for (let i = 0; i < len - 1; i++) {
    // 冒泡循环
    for (let j = i + 1; j < len; j++) {
      // 每次冒泡比较循环
      if (array[j] > array[i]) {
        // 交换位置
        let temp = array[i];
        array[i] = array[j];
        array[j] = temp;
      }
    }
    //console.log("每次冒泡后顺序", array);
  }
  return array;
}
```

### 3、优点

每进行一次排序，都会少比较一次，减少了算法的量

### 4、时间复杂度

O(n^2)
